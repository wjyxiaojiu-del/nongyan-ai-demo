## 🎯 项目简介

农眼AI 是一款面向设施农业种植户的**AI智能种植助手**，通过多模态大模型技术（视觉识别 + 知识图谱 + RAG检索增强）为农户提供病虫害识别、水肥精准推荐、环境预警、农技百科等一站式智能决策服务。

本项目是产品经理求职Demo，聚焦于农业+AI垂直领域，从需求调研、产品设计到交互原型的完整实践。

## ✨ 核心功能

### 1. 🤖 AI拍照识别
- **3秒识别**：拍摄或上传作物叶片照片，AI自动识别病虫害类型
- **置信度展示**：可视化展示识别准确率，低置信度自动触发专家复核流程
- **用药方案**：基于知识图谱生成精准用药方案（药剂名称、剂量、用法、安全间隔期）
- **识别记录**：自动保存历史识别记录，支持状态追踪（处理中/已解决）

### 2. 💧 水肥精准推荐
- **作物全周期覆盖**：支持番茄、黄瓜、辣椒、茄子、水稻等5种作物
- **生长期智能匹配**：根据所选生长期（苗期/开花期/结果期/采收期）动态生成差异化方案
- **真实农业数据**：基于水肥一体化技术文献，包含N-P-K配方、亩用量、稀释倍数、施用方式
- **土壤数据监测**：pH值、有机质、碱解氮、有效磷等指标实时展示

### 3. 🌡️ 环境预警
- **IoT数据接入**：温度、湿度、光照、CO₂浓度实时监测
- **智能阈值告警**：超温、高湿、异常光照自动推送预警
- **趋势图表**：24小时温度变化曲线，辅助环境调控决策

### 4. 📖 种植百科（百度搜索直连）
- **快捷搜索**：内置常见病虫害防治、栽培技术快捷入口
- **百度搜索集成**：一键跳转百度搜索结果，获取权威农技资料
- **热门文章推荐**：基于搜索热度的农技文章列表

## 🛠️ 技术栈

| 技术 | 说明 |
|------|------|
| HTML5 + CSS3 | 纯前端单页应用（SPA），无后端依赖 |

## 🚀 快速开始

```bash

# 方式2：启动本地服务器
```

2. 进入 Settings > Pages
3. Source 选择 Deploy from a branch
4. Branch 选择 master / (root)
5. 等待1-2分钟，访问 `https://<your-username>.github.io/nongyan-ai-demo/`

## 📊 产品方法论体现

| 方法论 | 应用体现 |
|--------|----------|
| **KANO模型** | 基础型（识别准确率）、期望型（用药方案）、兴奋型（AI语音播报） |
| **AARRR模型** | 获取（农技短视频引流）→激活（拍照即识别）→留存（识别记录）→收入（专家付费咨询）→推荐（分享裂变） |
| **MVP思维** | 单HTML文件实现核心闭环，验证最小可行产品 |
| **用户分层** | 针对中老年种植户的大字体、大按钮、语音播报适老化设计 |
